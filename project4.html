Purrfect Pair

Role:

Lead Game Designer and Developer

Date:

December 2023

Context:

University of Colorado Boulder

















Description:

  "Purrfect Pair" is a cat-themed game app combination, featuring two simple games—a yarn-collecting game and a matching game—within one app. It also tracks and displays the user’s high scores after each play. The primary goal of this project was to introduce myself to game development for Apple devices, using Swift (more specifically, SpriteKit).

This game does have a code base you can reference on GitHub.
