Singing Cylinders

Role:

Lead Object Designer, 3D Modeler, and Renderer

Date:

March 2022

Context:

University of Colorado Boulder

















Description:

  "Singing Cylinders" is an interactive sculpture installation inspired by Paul Matisse’s "The Musical Fence". This project highlights both the artistic and technical aspects of user interaction, while showcasing my ability to create accurate, true-to-scale renderings. The installation was sculpted using AutoDesk Fusion and 3D rendered in Adobe Dimension and was a project for one of my core classes.
